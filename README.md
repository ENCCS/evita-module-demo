## Contributing

To contribute to Gitlab, first create a [Personal access token](//-/profile/personal_access_tokens).
Then either `git clone`

```sh
git clone https://<username>:<personal_token>@gitlab.bsc.es/evita/module-template.git
```

or if you have an existing repository use `git remote set-url origin`:

```sh
git remote set-url origin https://<username>:<personal_token>@gitlab.bsc.es/evita/module-template.git
```

For bigger features, always make changes via a merge-request.
