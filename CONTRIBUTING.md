# Contributing

To contribute to Gitlab, you can either create an [SSH key](https://code.europa.eu/-/user_settings/ssh_keys) or a [Personal access token](https://code.europa.eu/-/profile/personal_access_tokens).

## SSH key

Download the repository using `git clone`

```sh
git clone git@code.europa.eu:eurohpc-ju/evita/module-template.git
```

or if you have an existing repository use `git remote set-url origin`:

```sh
git remote set-url origin git@code.europa.eu:eurohpc-ju/evita/module-template.git
```

## Personal access token
Download the repository using `git clone`

```sh
git clone https://<username>:<personal_token>@code.europa.eu/evita/module-template.git
```

or if you have an existing repository use `git remote set-url origin`:

```sh
git remote set-url origin https://<username>:<personal_token>@code.europa.eu/evita/module-template.git
```

## Access to the project

You can contact [one of the owners](https://code.europa.eu/groups/eurohpc-ju/evita/-/group_members) to add you to the project.

## Submit changes and suggestions

Start by [creating an issue](https://code.europa.eu/eurohpc-ju/evita/module-template/-/issues) to discuss your idea.

For bigger features, always make changes via a merge-request.
